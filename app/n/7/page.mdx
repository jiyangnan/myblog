# Moltbot 打通 Telegram Bot，构建个人 AI 助手

<div className="article-date">2026.01.29</div>

# 安装

输入如下指令：

```
curl -fsSL https://molt.bot/install.sh | bash
```

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129195258815.png)

进来以后系统就会提示你这玩意儿很危险，要不要继续。那肯定得继续啊，越危险越兴奋！

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129195451452.png)

第一次先让它自动设置一下

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129195634230.png)

选择一个模型

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129201911542.png)

先用 codex 试一下

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129202032573.png)

接下来会提示一下 channel status 以及 channel 的工作方式。

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129202141948.png)

那我们首先就用 TG 来试一下：官方给的说明：

1) Open Telegram and chat with @BotFather
2) Run /newbot (or /mybots)
3) Copy the token (looks like 123456:ABC...)
Tip: you can also set TELEGRAM_BOT_TOKEN in your env.
Docs: https://docs.clawd.bot/telegram
Website: https://clawd.bot

我们实操一下。

# 2. 配置 Telegram Bot

1. 首先 @BotFather

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129203834625.png)

然后点击打开，随意取一个名字，记得以 "bot" 结尾。创建：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129204122194.png)

然后把上面这一串 key 复制到 ClawdBot，回车：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129204329709.png)

之后他就会问你要不要装一些 skills，装，必须装：

我暂时先装这 4 个：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129204648177.png)

之后会去问你要不要创建一些 API Key，我这里只选择了配置 Gemini API KEY

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129212318742.png)

之后把这三个 hooks 都勾上。不选也没有关系，之后可以通过如下指令另行配置。

```
clawdbot hooks list
clawdbot hooks enable <name>
clawdbot hooks disable <name>
```

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129212502348.png)

此配置完毕。

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129212907087.png)

# 3. 调试 Telegram Bot

首先我们 GUI（图形化界面）试和他沟通：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129214226013.png)

然后先给他一个人设：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129214647246.png)

然后去 TG，找到刚才创建的机器人，/start，他会发送 1 个 ID，还有配对码。

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129215219618.png)

然后告诉 ClawdBot 这两个信息，让他去配置：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129221338646.png)

然后就大概等待一分钟左右就会收到机器人的推送：

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129221246404.png)

很棒，我们来看一看他能干点什么。

然后神奇的事情就发生了，你会发现给他说很多的话，但是他压根儿不回复。

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129231045959.png)

原因是，虽然配对成功了，但是并没有批准 telegram 给他发消息，需要让他去同意一下：

```
approve telegram {配对码}
```

然后就好了

![image.png](https://raw.githubusercontent.com/jiyangnan/picBed/main/img/20260129231453064.png)

至此完成！

开始玩！
